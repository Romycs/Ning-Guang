# 🔥 Нин Гуан - Telegram Video Downloader Bot

Мощный Telegram бот для скачивания видео и аудио с популярных платформ.

## 🎯 Возможности

- 🎬 **Скачивание видео** с YouTube, TikTok, Instagram, Facebook
- 🎵 **Скачивание аудио** в формате MP3
- 📥 **Массовое скачивание** - загрузка нескольких ссылок одновременно
- 📝 **История** - просмотр последних 10 скачиваний
- 🗑️ **Очистка истории** - удалить всю историю скачиваний

## 📋 Требования

- Node.js 14.0.0 или выше
- npm или yarn
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

## 🚀 Установка

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Romycs/Ning-Guang.git
cd Ning-Guang
```

### 2. Установите зависимости
```bash
npm install
```

### 3. Создайте файл .env
```bash
cp .env.example .env
```

### 4. Добавьте свой Bot Token
Отредактируйте файл `.env` и вставьте ваш токен:
```
BOT_TOKEN=YOUR_BOT_TOKEN_HERE
```

## 🎮 Использование

### Запуск бота
```bash
npm start
```

### Запуск в режиме разработки (с автоперезагрузкой)
```bash
npm run dev
```

## 📱 Команды Telegram бота

| Команда | Описание |
|---------|---------|
| `/start` | Начать работу с ботом |
| `/history` | Просмотреть последние 10 скачиваний |
| `/clear` | Очистить историю скачиваний |
| `/done` | Завершить режим мульти-ссылок |

## ⚠️ Важные замечания

1. **Лицензия**: Убедитесь, что у вас есть право скачивать контент с выбранных платформ
2. **API**: Некоторые API могут быть недоступны или требовать обновления
3. **Размер файла**: Telegram имеет ограничения на размер файлов (до 2GB)

## 🐛 Поиск и исправление ошибок

### Ошибка: "BOT_TOKEN не установлен"
**Решение**: Убедитесь, что файл `.env` создан и содержит корректный токен

### Ошибка: "Cannot download video"
**Решение**: Проверьте:
- Корректность ссылки
- Доступность видео (не приватное)
- Работоспособность API

### Ошибка: "ENOTFOUND api.tiklydown.me"
**Решение**: API может быть недоступен, попробуйте позже или используйте другой источник

## 🔐 Безопасность

- ✅ BOT_TOKEN хранится в `.env` и не коммитится в Git
- ✅ История хранится в памяти (теряется при перезагрузке бота)
- ✅ Каждый пользователь имеет отдельную историю
